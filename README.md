## `⬡` CE QUE JE CONSTRUIS

Je conçois des **systèmes backend robustes et durables** — APIs fiables, architectures propres, code pensé pour évoluer avec le produit sans dette technique accumulée.

Mon cœur de métier est l'écosystème **Java / Spring Boot**, structuré autour de patterns d'architecture éprouvés qui gardent la complexité sous contrôle à mesure que les projets grandissent.

> 💡 Ingénieur indépendant et fondateur de **[MV-TECH](https://mv-tech.vercel.app)** — un studio orienté conception de produits numériques bien architecturés.

## `⬡` APPROCHES ARCHITECTURALES

<div align="center">

| Pattern | Description |
|:---:|:---|
| 🧱 **Monolithe Modulaire** | Base solide pour les produits en croissance — découplage interne sans complexité opérationnelle |
| 🔩 **Microservices** | Scalabilité indépendante, déploiements isolés, résilience par domaine |
| 🔷 **Architecture Hexagonale** | Isolation du domaine métier, testabilité maximale, ports & adapters |
| 📐 **DDD** | Modélisation orientée métier, langage ubiquitaire, cohérence forte du domaine |

</div>
